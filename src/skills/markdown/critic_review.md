# Critic Review

## When to activate
- Reviewing implementation quality before release.

## Workflow
1. Validate requirements coverage.
2. Identify defects and risks.
3. Emit must-fix items with severity.

## Rules
- Every issue must include impact and remediation hint.

## Output format
- Return structured issues and blocking verdict.

## Failure conditions
- Missing severity tags.
- No actionable remediation details.
