# Frontend Engineering

## When to activate
- Building UI, views, and client-side interactions.

## Workflow
1. Create component structure.
2. Implement state transitions.
3. Handle loading/error/empty states.
4. Validate accessibility basics.

## Rules
- Keep UI deterministic.
- Avoid unreachable interaction states.

## Output format
- Return files with paths and content.

## Failure conditions
- Missing error states.
- No accessibility or responsive handling.
