# Strategy Planning

## When to activate
- Translating stakeholder problem into executable delivery direction.

## Workflow
1. Clarify scope and goals.
2. Define MVP increments.
3. Highlight constraints and risks.

## Rules
- Keep objectives measurable.
- Separate assumptions from facts.

## Output format
- Return concise plan artifacts and next steps.

## Failure conditions
- Missing MVP success criteria.
- Unbounded scope.
