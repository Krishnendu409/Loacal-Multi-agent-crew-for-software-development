# Roadmap

## MVP (now)

- Stable local multi-agent orchestration and handoff workflow
- Configurable role routing/fallback model policy
- Skills framework with optional external ECC pack
- Output report generation and per-agent artifacts
- Baseline tests for agents, config, crew orchestration, skills, and tasks

## Reliability (next)

- CI enforcement for tests, linting, and type checks
- Strong configuration validation with clear startup errors
- Guardrails for unsafe model output display and persistence
- Runtime manifest and metrics per crew run
- Resumable execution from selected role

## Scale (later)

- Plugin-based role/task registration for extensibility
- Structured machine-readable role outputs for tighter downstream parsing
- Optional cited research mode for evidence-backed planning
- Extended integration and snapshot test suites
